# @tourcompanion/ios

Capacitor 6 wrapper around `@tourcompanion/web`. The web SPA is bundled into
`www/` and loaded by a native iOS shell generated by Capacitor.

## Dev loop

```
npm run build:web    # bundle core + copy packages/web/public -> www/
npm run cap:sync     # sync www/ into the iOS app
npm run cap:open     # open Xcode (manual run / debug)
npm run build:ios    # full headless build (CI / scaffold proof)
```

## Identity (locked)

- Bundle ID: `com.cloudaipro.tourcompanion`
- App name: `TourCompanion`

## What's committed

- Swift sources, `Podfile`, `Podfile.lock`, `project.pbxproj` under `ios/App/`
- `capacitor.config.ts`, `copy-web.mjs`, `package.json`, this README

Generated artifacts (not committed): `www/`, `node_modules/`, `ios/App/Pods/`,
`ios/App/build/`, `ios/App/DerivedData/`, `build/`.
